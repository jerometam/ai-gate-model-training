Teachable Machine Hovercraft 偵測測試

請貼上你的 Teachable Machine 模型網址（必須有兩個類別：Hovercraft 與 No Hovercraft）

如何建立並取得模型網址（點擊展開）
  1. 打開 Teachable Machine 網站。
  2. 選擇「影像專案 (Image Project)」。
  3. 建立兩個類別：Hovercraft 與 No Hovercraft。
  4. 在「Hovercraft」類別中，上傳或拍攝包含氣墊船的照片。
  5. 在「No Hovercraft」類別中，上傳或拍攝不包含氣墊船的照片。
  6. 完成影像收集後，點擊「訓練模型 (Train Model)」。
  7. 等待訓練完成，然後點擊「匯出模型 (Export Model)」。
  8. 選擇「TensorFlow.js」格式，並複製提供的模型網址。
  9. 將該網址貼到右邊的輸入框中，然後按下「開始模型」。

這樣就能在本頁面使用你訓練好的模型進行偵測。
